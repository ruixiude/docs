# 总线服务

Group ID Artifact ID
com.ruixiude.core bus-api

基于 RxJava 的事件总线。

# 使用

建议通过注入或其他适当的机制来获取总线的单个实例。

或者像下面这样获取单例：

val bus = RXDBus.instance
1

# 订阅

要订阅事件，请使用 listen() 监听方法订阅事件。

val testSubscriber = TestSubscriber.create<BusEvent>()

RXDBus.instance.listen().subscribe(testSubscriber)
1
2
3

还可以根据事件类型按照以下方式创建订阅：

val busEvent1 = object : BusEvent {
    override fun getId(): String = "test-id-1"

    override fun getType(): String = "test-1"

    override fun getData(): Map<String, String> = emptyMap()
}
    
val testSubscriber = TestSubscriber.create<BusEvent>()

RXDBus.instance.listenByEventType(busEvent1.getType())
    .subscribe(testSubscriber)

# 注册和取消注册

订阅事件后，通过订阅者自行管理事件的取消：

val testSubscriber = TestSubscriber.create<BusEvent>()
// 注册
RXDBus.instance.listen().subscribe(testSubscriber)
// 取消注册
testSubscriber.cancel()

# 添加拦截器

通过初始化方法可添加多个事件拦截器：

// 定义日志输出拦截
val busLoggerInterceptor = object : BusInterceptor {
    override fun onSend(event: BusEvent): Boolean {
        println("send    -----> id: ${event.getId()}")
        return false
    }

    override fun onReceive(event: BusEvent): Boolean {
        println("receive <----- id: ${event.getId()}")
        return false
    }
}
// 定义事件拦截
val busInterceptor = object : BusInterceptor {
    override fun onSend(event: BusEvent): Boolean = event.getId() == "test-id-3"

    override fun onReceive(event: BusEvent): Boolean = false
}
// 初始化
RXDBus.init(arrayListOf(busInterceptor, busLoggerInterceptor))
